Web Variables Component Demo

A powerful custom element for managing state across your web application

Introduction

The Web Variables component allows you to manage application state with a simple HTML-based syntax. It supports multiple storage backends and provides tools for reacting to state changes.

Storage Type

Current storage: localstorage

JavaScript API

You can also interact with variables directly through JavaScript.

Direct Variable Access

Try these in your console:

// Get the web-variables element
const webVars = document.querySelector('web-variables');

// Set a variable
webVars.setVariable('username', 'JohnDoe');

// Get a variable
webVars.getVariable('username');

// Watch for changes
webVars.watchVariable('username', (newVal, oldVal) => {
    console.log(`Username changed from ${oldVal} to ${newVal}`);
});
